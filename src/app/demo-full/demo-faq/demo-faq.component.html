You can find more details of the component, various inputs and outputs in the component's original repository .

Note :

The width of the overall Faq container is fluid in nature i.e, 100% of the parent conatainer, so user can control the width of the grid based on the parent element in which the card is contained.

Text inside this component is not restricted to particular language. Based on your i18n preferences pass inputs in the desired language.
