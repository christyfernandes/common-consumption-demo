Card Stack

Library Card Stack Component

Library Card Stack Component can be used in the library page or in places where a card Stack needs to shown.

Stack is meantn to be used in mobile client. This component internally uses LibraryCard component with type='mobile_textbook'

Basically there is only 1 type of Stack:

  • Default Stack - Stack of mibile cards with option to specify maximum cards(i.e, 3, 4 or 5) to be shown in a row and a view all button, Default maximum cards count is 3

You can find more details of the component, various inputs and outputs in the component's original repository .

Note :

The width of the overall Stack is fluid in nature i.e, 100% of the parent conatainer, so user can control the width of the Stack based on the parent element in which the card is contained.

Card Stack is not restricted to particular language. Based on your i18n preferences pass inputs in the desired language.

In offline/online/downloaded scenarios where image attribute changes based on the network or other criteria, cardImg property must be updated to all the cards in the list.
