Library Card Component

Library Card is the component that is meant to be used in the places where a text book or content needs to be shown in the card style.

This component can be used in library section across all the consumption platforms

Basically there are four types of cards:

  • Recently Viewed card
  • Mobile view textbook/content
  • Desktop view textbook/content
  • QR code results textbook/content

You can find more details of the component, various inputs and outputs in the component's original repository .

Note :

The width of the card is fluid in nature i.e, 100% of the parent conatainer, so user can control the width of the card based on the parent element in which the card is contained.

Card is not restricted to particular language. Based on your i18n preferences pass inputs in the desired language.

In offline/online/downloaded scenarios where image attribute changes based on the network or other criteria, cardImg property must be updated.
