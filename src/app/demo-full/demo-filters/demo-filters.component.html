Filters

Filters Component

Library filters Component can be used in the library page, toc page and in all places where filters/pills need to shown

Basically there are 3 types of Filters based on the style:

  • Sqaure Filters
  • Round Filters
  • Round Filters With Icon

You can find more details of the component, various inputs and outputs in the component's original repository .

Note :

The width of the overall Filters container is fluid in nature i.e, 100% of the parent conatainer, so user can control the width of the grid based on the parent element in which the card is contained.

Text inside filter/pills is not restricted to particular language. Based on your i18n preferences pass inputs in the desired language.

Sqaure Filters

For default selection of any filter/pill pass the index of required item. eg:[selectedItems] = [1]

Round Filters

For default selection of any filter/pill [selectedItems] = [index number] pass the index of required item.

Round Filters With Icon

For default selection of any filter/pill [selectedItems] = [index number] pass the index of required item.
